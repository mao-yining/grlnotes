# 语法修辞逻辑精讲（Grammar Rhetoric Logic Notes）

使用 LaTeX 重新排版，并做了一些修订。

## 下载

本项目暂时没有 release 版本，但是每一次 push 后的自动构建，使用思源字体。可以在 Actions 界面下的 Artifacts 处下载。

## 依赖

使用 XeLaTeX 编译本项目。用到的宏包列表如下：

- ctex
- ifplatform
- geometry
- graphicx
- longtable
- bigstrut
- multirow
- multicol
- tocbibind
- ulem
- biblatex
- ifdraft
- imakeidx
- hyperref

## Contributing

当然，语文的这些知识大部分是从各大教科书中摘录而来。如果你对本项目有想法，欢迎提交 PR 或 issue。
